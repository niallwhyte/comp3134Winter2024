Web server ports are the logical endpoints of a network connection that is used to exchange information between a web server and a web client.

Port numbers range from 0 to 65536, but only ports numbers 0 to 1024 are reserved for privileged services and designated as well-known ports.

It represents a 16 bit value. 

10 common server ports: 80, 443, 21, 25, 110, 143, 22, 23, 53, 3389
